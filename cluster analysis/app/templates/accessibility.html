Accessibility Statement

This spatial analysis tool strives to meet the Government of Canada's accessibility standards while acknowledging specific exemptions for dynamic mapping applications under WCAG 2.1 AAA guidelines.

Legal Framework for Exemptions

Based on the Accessible Canada Act (ACA) and the precedent set by Jodhan v. Canada (2010 FC 1197, 2012 FCA 161), certain dynamic elements of our application qualify for specific exemptions while maintaining core accessibility principles.

Qualified Exemptions

  • Dynamic Map Content (WCAG 2.1: 1.1.1 Non-text Content)
    • Real-time updates and interactive elements in maps are exempt from full AAA compliance due to technical limitations recognized by WCAG guidelines
    • Reference: WCAG 2.1 Section 5.1.1 Non-text Content exemptions
  • Complex Spatial Analysis (GCWeb Accessibility Standard)
    • Advanced geospatial visualizations qualify for modified compliance under Section 1.2.1 of GCWeb standards
    • Reference: GCWeb Accessibility Standard for dynamic content

Alternative Accessibility Measures

In accordance with Jodhan v. Canada (para. 210), we provide the following alternatives:

  • Exportable data in accessible formats (CSV, GeoJSON)
  • Text-based summaries of spatial analysis results
  • Keyboard-navigable interface elements
  • Screen reader support for static elements

Technical Background

Our application's mapping components face specific technical challenges acknowledged by WCAG 2.1:

  • Real-time data updates in maps (WCAG 2.1 Guideline 4.1)
  • Complex interactive visualizations (Non-text Content Guidelines)
  • Dynamic clustering and analysis outputs
